# Residential Rental Market Analysis

## Етап 1: Збір даних.
Спочатку було вирішено зібрати дані з сайту `dom.ria`. Після кількох написаних тестових парсерів дійшов до найкращого і зібрав 1000 оголошень. Далі спробував очистити дані і розподілити по колонках - зрозумів, що не вистачає деяких даних і треба дописати парсер. Додав все необхідне в парсер і буду збирати ще 100 оголошень для тесту очистки і розподілення по колонках.
